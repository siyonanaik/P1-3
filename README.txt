Financial Trend Analysis Dashboard
This dashboard is a collaborative project for analyzing live financial data, including stock prices, RSI, and other key indicators.

Prerequisites
To run this application, you need to have Python 3.6 or higher installed on your system.

Installation
Navigate to the project directory in your terminal or command prompt.

Install the required libraries using the requirements.txt file. This command will install Streamlit, pandas, yfinance, and other dependencies.

""" pip install -r requirements.txt  """


Running the Application
Once all dependencies are installed ,run the Streamlit application from your terminal:

streamlit run app.py

This will launch a local web server and open the dashboard in your default browser.

Usage
Use the menu in the sidebar to navigate between different dashboards for stock analysis.
